## Proje Amacı

Bu proje, Cisco Networking Academy tarafından sunulan **Introduction to Cybersecurity** eğitimi kapsamında hazırlanmıştır.

Projenin amacı; siber güvenliğin temel kavramlarını, siber tehditleri, güvenlik açıklarını ve alınabilecek güvenlik önlemlerini tanıtmaktır.

Bu projede aşağıdaki konular ele alınmıştır:

- Siber güvenlik nedir?

- Kimlik Avı (Phishing)
